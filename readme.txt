To build Enchilada, you will need Maven installed on your machine: https://maven.apache.org/download.cgi

After installing Maven, you can clone this repo and use Maven from the command line to build Enchilada.
Basic instructions for how to use Maven can be found here: https://maven.apache.org/guides/getting-started/maven-in-five-minutes.html
You can ignore the part about creating a new project; the POM for Enchilada is included in the repo.
